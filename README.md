# ab-testing-analysis

📊 A/B Testing Analysis (Conversion Rate Optimization)

📌 Project Overview
This project analyzes the effectiveness of two versions (A and B) of a system to determine which performs better in terms of conversion rate.
The analysis is performed using statistical hypothesis testing.

🎯 Objective
To determine whether Version B improves conversion rates compared to Version A using a data-driven approach.

🧩 Dataset
The dataset contains:

User ID
Group (A = old version, B = new version)
Converted (0 = No, 1 = Yes)

Total Records: 400 users

🛠️ Tools & Technologies
Python (Pandas)
Statistical Testing (Z-test)
Matplotlib
Google Colab

📊 Key Analysis
Conversion Rate:
Group A: Lower conversion rate
Group B: Higher conversion rate
Hypothesis Testing:
Test Used: Proportions Z-test
P-value: 0.0108

✅ Conclusion
Since the p-value is less than 0.05, the result is statistically significant.
Version B performs better and should be implemented.

🚀 Key Learnings
Understanding of A/B Testing
Statistical significance using hypothesis testing
Data-driven decision making
